# Kenza Byte Member Adds for Telegram !

A simple automation tool by Kenzabyte that helps scrape and add Telegram group members using the Telegram API, designed for community growth and bulk member management.

Collect publicly available members from active Telegram groups and organize them for targeted community outreach and business growth. The tool helps identify relevant users from public groups, filter them based on activity and selected criteria, and streamline the management process efficiently from start to finish.
